
Purpose:
	This is a simple UART example in polling mode
	UART0 – 9600bps – 8 data bit – No parity – 1 stop bit – No flow control.

Hardware configuration:
	This example tested on Keil MCB1700 vers.1
	These jumpers must be configured as following:
		- VDDIO, VDDREGS, VBUS must be closed.

Running mode:
	This example can run on RAM mode with a debugger or run on FLASH.
	
Note:
	Print welcome screen first, then press any key to have it
	read in from the terminal and returned back to the terminal.
	- Press ESC to exit.
	- Press 'r' to print welcome screen menu again.
